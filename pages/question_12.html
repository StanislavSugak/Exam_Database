Запросы на языке SQL до своего реального выполнения подвергаются компиляции. Компиляция запросов может выполняться на стадии предкомпиляции прикладной программы, написанной на каком-либо языке программирования с предложением на языке SQL или в процессе выполнения транзакции с использованием инструкция языка SQL.

С точки зрения пользователя в процессе компиляции запроса получается следующий результат: для каждого предложения на SQL образуется программы в машинных кодах, вызовы которых помещаются в тексты сходной прикладной программы. Однако в действительности процесс распределённой компиляции запроса намного сложнее из-за наличия сетевых взаимодействий, которые требуется при реальных выполнения транзакций. Будем называть главным узлом сети тот узел, в котором инициализирован процесс компиляции предложений SQL и дополнительными узлами. Те узлы, которые включается в этот процесс в ходе его выполнения.

Процесс распределенной компиляции запроса можно разбить на следующие стадии:

1. В главном узле сети выполняется грамматический разбор предложения SQL с построением внутреннего представления запроса или дерева. На основе информации из локального каталога главного узла и удаленных каталогов дополнительных узлов выполняется замена имен объектов, фигурирующих в запросе, на их системные идентификаторы;

2. В главном узле генерируется глобальный план выполнения запроса, в котором учитывается только порядок взаимодействия узлов при реальном выполнения запроса. Глобальный план отображается в преобразованном соответствующем образом дереве запросов

3. Если в глобальном плане выполнения запроса участвуют дополнительные узлы, производится его декомпозиция на части, каждую из которых можно выполнить в одном узле.

4. В каждом узле участвующих в глобальном плане выполнения запроса в главном и дополнительном, выполняется завершающая стадия компиляции. Эта стадия включает оптимизация и генерацию машинных кодов. На этой стадии выполняется проверка прав пользователя от имени которого выполняется компиляция на выполнения соответствующих действий.
